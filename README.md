# Static Website Hosting on AWS S3 (AWS Console)

This project demonstrates hosting a static website on AWS S3 using the AWS Management Console.

## Overview
- Created an S3 bucket and uploaded static website files
- Enabled static website hosting from S3 bucket properties
- Configured bucket permissions to allow public access to website content
- Verified website accessibility using the S3 website endpoint

## Technologies Used
- AWS S3
- AWS Management Console

## Key Learnings
- Understanding of S3 static website hosting workflow
- Hands-on experience with bucket permissions and public access settings
- Basic knowledge of AWS-managed static content delivery

## Note
This project was performed manually using the AWS Console and serves as a foundational AWS learning exercise.
